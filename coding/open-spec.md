# 1.初始化
`openspec init`

# 2.创建提议(Proposal)，生成规范
```markdown
1. 填充你的项目上下文：
“请阅读 openspec/project.md 并帮助我根据我的项目、技术栈以及约定来完善其中的内容。”

2. 创建你的第一个变更提案：
“我想添加【在此填写你要的功能】。请为这个功能创建一份 OpenSpec 变更提案。”

3. 学习 OpenSpec 工作流程：
“请向我解释 openspec/AGENTS.md 中的 OpenSpec 工作流程，以及我应该如何与你一起协作完成这个项目。”
```

# 3.创建功能提议
`/openspec:proposal [question]`

# 4.实施开发
`/openspec:apply [question/task]`

# 5.归档任务(Archive)
`/openspec:archive [question/task]`
