# Prompt Templates

Use these templates when the request is real but underspecified.

## Universal Draft Template

```text
Create a [image type] for [audience/use case].

Subject:
[Who or what is the main subject? Include identity, appearance, clothing, or key features.]

Scene:
[Where is the subject? Include environment, time of day, and background.]

Style:
[What visual direction should it follow?]

Composition:
[What framing, layout, or aspect ratio should it use?]

Text:
[Exact text that must appear, if any.]

Materials and lighting:
[Textures, surfaces, light quality, reflections, atmosphere.]

Constraints:
Keep [must-keep elements].
Do not change [protected elements].
Make sure [critical success criteria].
```

## Prompt Rewrite Template

```text
Goal: [what the user is actually trying to achieve]

Rewrite this idea as a stronger Nano-Banana prompt.
Use natural language, clear composition, and explicit constraints.

Prompt:
Create a [image type] for [use case]. Show [subject] in [scene].
Use a [style] direction with [lighting].
Place [element A] on the [left/right/top/bottom] and [element B] on the [left/right/top/bottom].
Render the text "[exact text]" clearly in a [font feel] style.
Keep [must-keep traits]. Do not change [protected details].
Output in [aspect ratio or format].
```

## Iteration Template

```text
Keep [working elements] exactly the same.
Change only [failing elements].
Preserve [identity, pose, framing, layout, or other constraints].
Update the image so it feels [new mood or quality target].
Do not alter [must-not-change elements].
```

## Infographic Template

```text
Create a clean [editorial / technical / whiteboard] infographic for [audience].
Summarize [topic or source] into clear sections: [section names].
Include [chart or diagram types].
Render all text clearly and legibly.
Use a [color and typography direction] style.
```

## Thumbnail Template

```text
Design a viral thumbnail for [platform/use case].
Use the person from Image 1 as the identity anchor and keep the face exactly the same.
Place the person on the [left/right] side with an expression of [emotion].
Place [secondary subject] on the [left/right] side.
Add [graphic element].
Render the title "[exact text]" in a bold, readable [font feel] style.
Use a [background] background with strong contrast.
```

## Editing Template

```text
Keep the original composition, subject identity, and camera angle exactly the same.
Change only [target change].
Match the new content to the existing lighting, texture, and perspective.
Do not alter [must-keep elements].
```

## Storyboard Template

```text
Create a [number]-part storyboard featuring [subject].
Keep the character identity, wardrobe, and art direction consistent across all frames.
Vary the camera angle, distance, and expression to create a narrative arc.
End on [final emotional or visual beat].
Generate one frame at a time in [aspect ratio].
```
