## 情感类型与手法映射

### 1) 情感类型识别

| 情感类型 | 识别线索 | 推荐意象方向 |
|---|---|---|
| 等待/思念 | 持续动作 + 无回应 | 时间、空间、距离感 |
| 放手/释然 | 痛苦转接受 | 转身、背影、离场动作 |
| 心痛/失去 | 假设句 + 失落身体感 | 残留物、空位、未完成 |
| 温暖/感动 | 平凡细节 + 陪伴 | 自然光感、贴身小动作 |
| 倔强/坚持 | 对抗语气 + 逆境 | 身体动作、路感、硬质意象 |
| 迷茫/成长 | 回望 + 领悟 | 路线、站点、阶段意象 |

### 2) 手法选择（建议总数 2-4 种）

| 情感类型 | 主要手法 | 辅助手法 |
|---|---|---|
| 等待/思念 | 反复、序列、对比 | 顶针、重叠 |
| 放手/释然 | 对比、设问、衬托 | 反复 |
| 心痛/失去 | 夸张、反问、衬托 | 通感、反复 |
| 温暖/感动 | 赋、比拟、排比 | 对偶 |
| 倔强/坚持 | 反复、夸张、排比 | 对比、反问 |
| 迷茫/成长 | 设问、序列、对比 | 衬托 |

### 3) 段落功能对应手法

| 段落 | 优先手法 | 目的 |
|---|---|---|
| Verse | 赋、兴、序列 | 建场与铺垫 |
| Pre-Chorus | 排比、递进、设问 | 升温与蓄力 |
| Chorus | 反复（必选）、夸张、对偶 | 爆发与记忆点 |
| Bridge | 对比、衬托、反问 | 视角转折 |
| Final Chorus | 反复变化、夸张升级 | 情感超越 |

### 4) 使用禁忌

- 不要超过 4 种主要手法。
- 不要在 Verse 提前把主题喊完。
- 不要让 Bridge 变成 Chorus 的复读。
- 不要为了“文艺”牺牲可唱性。
