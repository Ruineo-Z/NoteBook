# RSS Fetcher

A skill for fetching and monitoring RSS feeds.

## Features
-   Configurable feed list via `config/feeds.json`.
-   Time-based filtering (e.g., items from the last 24 hours).
-   Supports both RSS and Atom feed formats.

## Setup
Ensure Python 3 is installed. The script uses standard libraries and `curl`.

## Usage
Use the `/rss-check` command to fetch updates.
